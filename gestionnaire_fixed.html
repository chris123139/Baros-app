❓ Guide — Envoi SMS groupé via Notchpay ✕
Notchpay est une plateforme camerounaise d'envoi de SMS compatible MTN Mobile Money et Orange Money. Les SMS sont facturés en FCFA directement depuis votre solde Notchpay.
1
Créer un compte Notchpay
Rendez-vous sur notchpay.co → cliquez S'inscrire.
Renseignez votre email, votre nom et le nom de votre entreprise (ex: Bar Chez Moi).
Confirmez votre email via le lien reçu dans votre boîte mail.
2
Recharger votre solde
Dans votre tableau de bord Notchpay → Portefeuille → Recharger.
Choisissez MTN MoMo ou Orange Money et entrez le montant en FCFA.
💡 Tarif estimé : ~5 FCFA par SMS local (Cameroun)
3
Récupérer votre clé API
Dans Notchpay → Paramètres → Clés API.
Copiez la clé qui commence par sk_live_...
⚠️ Ne partagez jamais cette clé — elle donne accès à votre compte.
4
Configurer BarOS
Dans BarOS → Config → Intégrations :
• Collez votre clé API dans le champ Clé API Notchpay
• Entrez un nom expéditeur (11 car. max, ex: LeBarAmos) — c'est ce que vos clients verront à la place d'un numéro
• Cliquez Sauvegarder puis Tester la connexion
5
Envoyer votre première campagne
De retour ici → choisissez un segment (ex: tous, ou inactifs +14j).
Rédigez votre message (160 car. max). Utilisez les variables {prenom} pour personnaliser.
Vérifiez l'aperçu, puis cliquez Envoyer via Notchpay. L'envoi se fait automatiquement à tous les clients du segment qui ont un numéro.
⚠️ Format des numéros clients
Les numéros doivent être enregistrés au format international dans les fiches clients :
✅ +237 6XX XXX XXX  ou  237 6XXXXXXXX
❌ 06XX XXX XXX (format local sans indicatif — non reconnu)
Questions fréquentes
Mon solde Notchpay est suffisant pour combien de SMS ?
À ~5 FCFA/SMS : 1 000 FCFA = 200 SMS · 5 000 FCFA = 1 000 SMS.
Les clients reçoivent le SMS de quel numéro ?
Ils verront le nom expéditeur que vous avez configuré (ex: LeBarAmos), pas un numéro de téléphone.
Puis-je tester avant d'envoyer à tous ?
Oui — créez un client fictif avec votre propre numéro, puis faites un segment ciblé.
Le bouton "Tester la connexion" dit "Erreur réseau"
Vérifiez que votre connexion internet est active et que la clé est bien collée (sans espace avant/après).
